### Users
#### 1. Registration

**Description:**

**Request:**

- URI

  ```http
  POST /auth/api/register
  ```


- Body

  ```Json
  {
	"data": {
        "email": "example-dev@mail.sustc.edu.cn",
        "password": "xxx"
    }
}
  ```


**Response:**

- Status Code: 201

- Body

  ```Json
  {
      "data": {
          "password": "$2a$10$YxMUShEsVAi5.g5mkHqdyec9poresC7e.abx5LqY4AZ3mpvE8/WZO",
          "id": 32,
          "email": "example-dev@mail.sustc.edu.cn"
      }
  }
  ```

**Errors:**

- Email exists    

- Status Code: 404  

- Body:
```JSON
{
    "errors": {
        "title": "This email has already been registered",
        "detail": "can not insert into users,violate email unique constrain",
        "status": 400
    }
}
```

####  4. Update User Profiles

**Description:**

- Some infomation such as email, user balance **MUST** not be updated by this way!
- **Only** the fields that appears in the JSON body should be updated!

**Request:**

- URI

  ```http
  PATCH /api/users/{id}/
  ```
- Header
Authorization: Bearer+" "+token

- Body

  ```Json
  {
      "data": {
          "id": "xxx",
          "username": "xxx",
          "password": "xxx",
          "phone": 1234567890,
          "credit": 100
      }
  }
  ```


**Response:**

- Status Code: 200 OK

- Body
```Json
{
    "data": {
        "id": "xxx",
        "type": "user",
        "username": "xxx",
        "email": "example@xxx.com",
        "phone": 1234567890,
        "credit": 100,
        "balance": 100,
        "nickname":"xxx",
        "level":0,
        "avatar":"URL"
    }
}
```

**Errors:**

- Permission denied

- Status Code: 403
- Body
```JSON
{
    "timestamp": "2018-06-08T08:04:59.663+0000",
    "status": 403,
    "error": "Forbidden",
    "message": "Access Denied",
    "path": "/api/users/32/"
}
```
